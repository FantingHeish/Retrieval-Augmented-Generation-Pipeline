# RAG PoC: Retrieval-Augmented Generation QA System
This project demonstrates a Retrieval-Augmented Generation (RAG) pipeline using HuggingFace Transformers and FAISS, deployed in Google Colab.

## Features
- Natural language QA over custom documents
- Embedding using sentence-transformers
- Vector search via FAISS
- Integration with LLM for answer generation
